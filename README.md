# The North Pole Post

A nostalgic Christmas postcard creator where users can design custom holiday greetings using vintage-style frames, stickers, stamps, and personalized messages.

## Features

- **Interactive Postcard Editor**: Create personalized Christmas postcards with a drag-and-drop interface
- **Decorative Elements**: 
  - 3 festive frames to choose from
  - 13 Christmas-themed stickers (Santa, trees, gifts, ornaments, etc.)
  - 7 vintage postal stamps
- **Customization Options**:
  - Add and position personalized text messages
  - Drag items to reposition anywhere on the canvas
  - Resize stickers, stamps, and text using corner handles
  - Delete individual elements with hover controls
- **Atmospheric Design**:
  - Animated falling snowflakes
  - Interactive snow trail following cursor movement
  - Video background with optional audio
  - Vintage wood and paper textures
- **Export**: Download your finished postcard design

## Technologies Used

- HTML5
- CSS3 (with custom animations and effects)
- Vanilla JavaScript
- Google Fonts (Mountains of Christmas, Dancing Script, Quicksand, Gloria Hallelujah)

## Installation

1. Clone this repository:
```bash
git clone https://github.com/g-Keerthiii/The-North-Pole-Post.git
```

2. Navigate to the project directory:
```bash
cd The-North-Pole-Post
```

3. Open `index.html` in your web browser

## Usage

1. **Select Decorations**: Click on frames, stickers, or stamps in the left sidebar
2. **Position Elements**: Drag items around the postcard to arrange them
3. **Resize Items**: Hover over an item and drag the resize handle in the bottom-right corner
4. **Add Messages**: Type your message in the text area and click "Add Message"
5. **Remove Items**: Hover over any item and click the red X button to delete
6. **Clear Canvas**: Use the "Clear All" button to start over
7. **Download**: Click "Download Postcard" to save your creation
